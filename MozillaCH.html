Localizing FirefoxOS apps

Localizing FirefoxOS apps

  • for l10n: FirefoxOS app === standards based HTML5 app

  • I18N for the web traditionally done on the server side
  • with HTML5 and SPAs markup is often rendered on the client side
  • client-side i18n support is required

  • among others Mozilla provides a solution with webL10n

webL10n

  • Github: https://github.com/fabi1cazenave/webL10n

  • declarative localization
  • named variables instead of printf-like %s tokens
  • a simple and full-featured pluralization system
  • server-less language negotiation (think of offline webapps)
  • JavaScript API

l10n-Attributes

<section>
	<h1 data-l10n-id=""header-main"></h1>
	<p data-l10n-id=""welcome-note"></p>
	<button data-l10n-id=""button-register-label">
	</button>
</section>

Language-files

header-main = Besuche den FirefoxOS...

JavaScript API

				// get a text in the current language
				var message = document.webL10n.get("header-main");
// setting the language re-translates the DOM
				document.webL10n.setLanguage("de-CH");					
// translate a DOM fragment (eg. after rendering)
				document.webL10n.translate(
	document.querySelector("#sidebar");
);

event="onlocalized"


window.addEventListener('localized', function() {
	document.documentElement.lang = 
		document.webL10n.getLanguage();
	document.documentElement.dir = 
		document.webL10n.getDirection();
}, false);
